# imooc-api

## imooc-admin的后台接口

## 希望大家能够支持我们的项目，欢迎star，fork，和issue

## 框架版本

nodejs：v16.3.1
koa：v2.13.4

### 安装依赖

```shell
npm install
```

### 启动项目

```shell
npm run dev
```

### mongoDB数据库

- 在本地下载安装mongodb数据库 
  [下载地址](https://www.mongodb.com/download-center/community)

- 下载MongoDB Compass 可视化工具
  [下载地址](https://docs.mongodb.com/v3.2/tutorial/install-mongodb-compass/)

### 导入数据

/db/mongodb-data/下的json文件

或者直接运行项目
